Sharkduino

The bulk of the research I have done at William and Mary has been on the Sharkduino project. This project focuses on creating small low cost animal tags to be used in marine science research. I created the initial prototypes, code, and analysis algorithms for the project. Now the scope and size of the project has expanded and I am in charge of the hardware portion. Most of my work right now is trying to improve the battery life of the system, testing new components that may be added to the system, and creating PCB boards to be used.

For more information about the project please check out the blog I mantian about it located here. Additionally I wrote a paper and created a poster about the project at the end of my summer research, they can be found here and here, respectively. I have presented this poster at the W&M Physics REU poster competition, where it won first prize, and at the W&M Physicsfest, where it received an honorable mention

Finaly this project has been featured in multiple outlets. A story about it was published by W&M News, it can be found here. The National Science Foundation (NSF) featured a video about the project in Science360, it can be found here.
